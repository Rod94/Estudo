## Dicas do Terminal

Lista de comandos que estou aprendendo:

- `cd` (navega entre pastas)
  Exemplo para entrar dentro da pasta:

```
cd nomeDaPasta
```

Exemplo para sair de uma pasta:

```
cd ..
```

- `mkdir` (cria uma nova pasta)
  Exemplo para criar uma nova pasta:

```
mkdir nomeDaPasta
```
